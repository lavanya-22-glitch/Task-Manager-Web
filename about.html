PROJECT DESCRIPTION

Task Manager is a lightweight web application designed to help users manage their daily tasks efficiently. It was built as part of a personal portfolio project by Lavanya Bhadani, a student at IIT Patna.

It focuses on simplicity, clean UI, and essential features — making task tracking easy for students and professionals alike.

  • Tailwind CSS
